# Meow NFT Project 🐱🏆

Sui Testnet上で、東大ブロックチェーン講座のグループワーク優勝を記念したNFTを発行・配布するためのプロジェクトです。

## 📁 構成
- **`sources/`**: Sui Move言語によるスマートコントラクト（NFTの定義）
- **`distribute_meow.sh`**: 【ローカル専用】一撃でNFTを発行・配布するスクリプト（実行例付き）
- **`mint_nft.sh`**: 汎用的な配布用自動化スクリプト
- **`Move.toml`**: プロジェクトの設定と依存関係
