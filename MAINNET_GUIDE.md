# Mainnet Deployment Guide & Cost Estimate 🚀

このドキュメントでは、今回のプロジェクトを Sui Testnet から **Mainnet（本番環境）** へ移行する際の手順、費用、および注意点をまとめています。

## 💰 費用概算 (2025年12月時点)
SUI価格を 1 SUI = ¥230 と想定した概算です。

| 項目 | SUI量 (目安) | 日本円 (目安) | 備考 |
| :--- | :--- | :--- | :--- |
| **コントラクト公開 (Publish)** | 1.0 ~ 2.0 SUI | **¥230 ~ ¥460** | 一番最初の一度のみ必要 |
| **NFTミント (1枚あたり)** | 0.005 ~ 0.01 SUI | **¥1 ~ ¥3** | 配布する人数分必要 |
| **合計 (5名に配布する場合)** | 約 2.0 SUI | **約 ¥460** | 500円分あれば十分足ります |

## 🛠️ メインネット移行時の変更ポイント

### 1. Sui CLI の環境切り替え
ターミナルで接続先をメインネットに変更する必要があります。
```bash
# メインネットに切り替え
sui client switch --env mainnet

# もしメインネット設定がない場合は追加
# sui client new-env --alias mainnet --rpc https://fullnode.mainnet.sui.io:443
```

### 2. 本物の SUI の準備
テストネットの `faucet`（蛇口）は使えません。
- 取引所（Bybit, OKX, Binance等）またはウォレット内のブリッジ機能等で本物の SUI を入手し、自分のアドレス（`0xf7fbe...`）へ送金しておく必要があります。

### 3. スクリプト (`distribute_meow.sh`) の更新
メインネットへ移行すると **Package ID** が新しく発行されます。
- デプロイ後に表示される新しい `Package ID` をスクリプト内の変数に書き換えてください。
- また、SuiScanのURLも `testnet` から `mainnet` へ修正が必要です。

---

## ⚠️ 注意事項

1. **修正が不可能な「不変性」**: 
   メインネットに一度デプロイすると、そのコードは一生残ります。バグがあった場合、修正版を「再デプロイ」することはできますが、古い（間違った）NFTを消すことはできません。デプロイ前にテストネットで完璧に動作確認をしてください。

2. **ガス代の変動**: 
   Suiのガス代は非常に安定していますが、ネットワークが極端に混雑している時は、計算上の費用が少し上がる場合があります。念のため、予算の1.5倍程度のSUIを持っておくと安心です。

3. **画像CIDの恒久性**:
   メインネットで発行する際は、Pinataにアップロードした画像が「間違っていないか」「削除する予定がないか」を再確認してください。一度発行したNFTの画像URLを後から変更することは（今のプログラムの仕組み上）できません。
